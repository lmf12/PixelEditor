# PixelEditor

这是一个可以绘制并保存像素画的桌面应用。

### 一、界面

顶栏是常用操作栏；左侧边栏是工具栏，提供画笔、橡皮擦、吸管和油漆桶四种工具；右侧边栏是取色盘；中间是画布。
